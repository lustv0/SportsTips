# Sport Stats References

These documents are lightweight reference files for the SportsTips workflow.

## Purpose

- Store stable usage, role, injury-watch, and market notes by sport.
- Give the betting workflow a reusable summary instead of rebuilding every baseline from scratch.
- Keep the notes concise and update them when a new request materially changes the picture.

## Files

- `afl-reference.md`
- `mlb-reference.md`
- `nba-reference.md`
- `nfl-reference.md`
- `nhl-reference.md`
- `nrl-reference.md`
- `soccer-reference.md`
- `tennis-reference.md`

## Update Rule

- Refresh only the sport files relevant to the current request.
- Store repeatable role and opportunity notes, not random short-term noise.
- Keep each file compact enough to stay readable.
